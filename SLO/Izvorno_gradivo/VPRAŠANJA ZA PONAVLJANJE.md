**VPRAŠANJA ZA PONAVLJANJE**

**LJUDSKO SLOVSTVO IN ORIENTALSKE KNJIŽEVNOSTI**

Pesmi, ki jih bo na pisnem ocenjevanju potrebno (vsebinsko in/ali
oblikovno) razložiti, boste dobili kot prilogo.

1.  Kaj so glavne značilnosti ljudskega slovstva?

2.  Katere literarne vrste so prisotne v ljudskem slovstvu?

3.  Katere so prepoznavne slogovne značilnosti ljudskega slovstva?

4.  Kateri so bili prvi zapisovalci ljudskega slovstva pri nas?

5.  Katera institucija danes zbira ljudske pesmi?

6.  Kaj je snov pripovedke o Petru Klepcu?

7.  Kaj sta osrednja motiva pripovedke o Petru Klepcu?

8.  Kako imenujemo glavni dve varianti pripovedk o Petru Klepcu?

9.  Kaj je značilno za žensko balado?

10. Kaj je značilno za balado?

11. Kateri so glavni motivi v pesmi *Od lepe Vide*?

12. Katere so variante pesmi o Lepi Vidi?

13. Kako je pesem o Lepi Vidi vplivala na nadaljnjo slovensko umetnost?

14. Simbol česa je riba faronika?

15. Iz pesmi *Riba faronika* izpiši stilne figure in jih poimenuj.
    Katere od teh stilnih figur so značilne za ljudstvo slovstvo?

16. Kaj je tema pesmi *Trgaj mi rožice, delaj mi pušelček*?

17. Kaj je motiv pesmi *Trgaj mi rožice, delaj mi pušelček*?

18. Iz pesmi *Trgaj mi rožice, delaj mi pušelček* izpiši stilne figure
    in jih poimenuj. Katere od teh stilnih figur so značilne za ljudsko
    slovstvo?

19. Iz katerega časa izvirajo orientalske književnosti?

20. Katera izmed orientalskih književnosti je najstarejša in kdaj se je
    razvila?

21. Kateri izmed orientalskih književnosti sta najmlajši in kdaj sta se
    razvili?

22. Katere so glavne značilnosti egipčanske književnosti?

23. Katere so glavne značilnosti babilonsko-asirske književnosti?

24. Katere so glavne značilnosti starojudovske oz. starožidovske
    književnosti?

25. Katere so glavne značilnosti perzijske književnosti?

26. Katere so glavne značilnosti indijske književnosti?

27. Katere so glavne značilnosti arabske književnosti?

28. Katere so glavne značilnosti kitajske književnosti?

29. Katere so glavne značilnosti japonske književnosti?

30. Kako delimo *Biblijo*?

31. Kateri so najpomembnejši prevodi *Biblije*? Navedi, v kateri jezik
    je bila prevedena in kako se ta prevod imenuje.

32. V katerem delu Biblije najdemo *Priliko o izgubljenem sinu*?

33. Kdo naj bi govoril *Priliko o izgubljenem sinu* in komu?

34. Koga v *Priliki o izgubljenem sinu* predstavlja oče, koga mlajši sin
    in koga starejši sin?

35. Kaj je sporočilo *Prilike o izgubljenem sinu*?
